# Task Management Full App

## Overview
A comprehensive task management application built for organizing and tracking tasks efficiently.

## Features
- Create, read, update, and delete tasks
- Task categorization and filtering
- Priority levels and due dates
- User authentication
- Real-time updates

## Installation
```bash
git clone <repository-url>
cd "Task Mangment"
npm install
```

## Usage
```bash
npm start
```
